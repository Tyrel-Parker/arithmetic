# Math Practice

A simple single-file web app for elementary math practice, built for an 8-year-old.

## Features

- **Four operations:** addition, subtraction, multiplication, and division
- **Configurable difficulty:** set the maximum number for generated problems
- **Division options:** allow remainders (quotient + remainder entry)
- **Negative numbers:** optional toggle for any operation
- **Three practice modes:**
  - **Timed** — answer a set number of questions, then see results
  - **Infinite** — keep going until you choose to stop
  - **Running Average** — track your average time per question over an open session
- **On-screen calculator:** tap-based input, no keyboard required
- **Results summary:** total questions, total time, and average time per question
- **Themes:** 6 preset color themes plus custom color picker, persisted via localStorage

## Usage

Open `index.html` in any browser — no build step, no dependencies.
